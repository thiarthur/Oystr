Environment: This application uses PHP (Tested on 5.3), and requires cURL LIB (https://curl.haxx.se/download.html).

HTML Version: The first analysis will be made with Document Type definition, which indicates that the version is lower than version 5. In sequence, older and newer HTML elements will be listed for a better sinalization

Page Title: The content will be extracted from <title> tag

Links : The server will find <a> tags. For each element, the href attribute is going to get tested for http protocols (http:// or https://).
If the link contains any protocol, the domain will be also tested, eliminating the inner domain links.
If does not, it is inferred that the link is external.
